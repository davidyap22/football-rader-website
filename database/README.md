# Database Scripts

This folder contains SQL scripts for Supabase database setup and maintenance.

## Contents

### Payment System
- `create_payment_tables.sql` - Payment tables schema (X1PAG integration)

### News System
- `supabase_news_comments.sql` - News comments table schema
- `update_news_comments_table.sql` - News comments table updates

### User Management
- `supabase_user_subscriptions.sql` - User subscriptions table
- `supabase_user_predictions.sql` - User predictions table

### Performance & Analytics
- `supabase_performance_functions.sql` - Performance tracking functions

---

**Usage:** Run these scripts in your Supabase SQL editor to set up or update database tables.
